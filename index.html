🛠️ Quick Setup

  • Clone this repository: git clone https://github.com/kade-jyothi/task-management-application.git
  • Navigate to backend: cd backend
  • Install dependencies: npm install
  • Start backend server: npm start
  • Open frontend in browser: Navigate to frontend directory and open any HTML file
